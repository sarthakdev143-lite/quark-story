================================================================================
HOW TO BUILD THE .XNT (WHEN ENVIRONMENT IS AVAILABLE)
================================================================================

Prerequisites:
   1. Windows 10/11
   2. Visual Studio 2022 + C++ workload
   3. Quark XDK 2024 (contact Quark for partner access)
   4. CMake 3.20+
   5. vcpkg with libcurl installed

Build Commands:
   cd XTension
   mkdir build && cd build
   cmake .. -G "Visual Studio 17 2022" -A x64 ^
            -DCMAKE_TOOLCHAIN_FILE=C:/vcpkg/scripts/buildsystems/vcpkg.cmake
   cmake --build . --config Release

Output: XTension/build/Release/QuarkStoryXT.xnt

Deployment:
   Copy QuarkStoryXT.xnt to:
   C:\Program Files\Quark\QuarkXPress 2024\XTensions\
